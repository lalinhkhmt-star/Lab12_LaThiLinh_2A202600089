# Screenshots

Thư mục này hiện chứa screenshots local verification được render tự động từ kết quả test thật:

- `stack-status.png`: trạng thái Docker stack và image size
- `service-running.png`: trạng thái service, health, readiness và sample response
- `api-test.png`: auth, rate limit và stateless verification
- `dashboard.png`, `running.png`, `test.png`: alias tên file để khớp format checklist nộp bài
